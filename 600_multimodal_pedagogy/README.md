# Multimodal Pedagogy (600)
**Systemic Dignity Infrastructure Audio & Visual Syndication**

This directory contains the syndication assets, scripts, and deployment manifests required to translate academic working papers and policy documents into accessible, multimodal formats.

## Architecture
The pedagogical approach utilized here is **Organic Vernacular Pedagogy (OVP)**. This methodology explicitly rejects traditional institutional authority and academic credentialism in favor of direct, lived-experience translation. 

### Core Principles
1. **Body-First Sequencing**: All theoretical concepts (e.g., identity capital, return deficits) are translated first through physical, biological realities (starvation, sleep deprivation).
2. **Subpopulation Taxonomy**: Avoids the generalized label of "homelessness," instead producing distinct audio/visual tracks tailored to specific structural failure modes (Acute Dislocation, Chronic Collapse, Institutional Cycling, etc.).
3. **Falsifiability**: Modules clearly state the structural conditions under which their own theories fail.

### Subdirectories
- `ovp_audio_scripts/`: Markdown scripts for the OVP podcast tracks, organized by subpopulation taxonomy.
- `manifests/`: JSON syndication manifests for automating multi-platform deployment.
