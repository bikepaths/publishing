# 05_working_dignity

This directory contains the fifth working paper in the Material Dignity Infrastructure series. It theorizes Economic Dignity Infrastructure (EDI), focusing on the agentive selfhood layer and structural pathways for economic reintegration.

## Hierarchy
* **01_drafts**: Raw text generation, manuscript sections, and iterative revisions.
* **02_research**: Economic source data, labor market analyses, and external citations.
* **03_outlines**: Structural blueprints mapping the progression from MDI to RDI to EDI.
* **04_ssrn**: LaTeX compiled output, metadata, and submission-ready formatting.
