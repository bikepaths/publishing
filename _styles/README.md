# Publishing Styles Decision Matrix

This directory contains the Manuals of Style (MoS) for the Full Spectrum Publishing Pipeline. 

**MUTUAL EXCLUSIVITY MANDATE:** Only one MoS may be active for a given document. Never combine constraints from multiple style guides.

Use this decision matrix to select the correct active MoS based on document intent.

---

## 1. The Empathetic Community Voice
**Use:** `MoS_OVP_Organic_Vernacular_Pedagogy.md` (Organic Vernacular Pedagogy)
**Target:** General community, story-driven lessons, shared experiences.
**Characteristics:** Grounded physical metaphors, flowing causality, empathetic witness tone. No sentence length caps. Rejects cold academic detachment. Focuses on the observable physical experience.

## 2. The Analytical Observation Voice
**Use:** `MoS_Systemic_Analysis.md`
**Target:** Sociological breakdowns, systemic friction analysis, detached observation.
**Characteristics:** Cold, objective, authoritative. Rejects narrative avatars, emotion, and rhetorical filler. Uses jagged paragraph architecture and conceptual terminology.

## 3. The Instructional Scaffolding Voice
**Use:** `MoS_Pedagogical_Docs.md`
**Target:** High school level ESL students requiring direct technical instruction.
**Characteristics:** Strict vocabulary limits, progressive concept scaffolding (concrete before abstract), explicit causal connectors.

---

## Formatting vs. Voice
Stylistic voice is distinct from publication formatting. 
Refer to `Publication_Formats.md` for specific technical requirements regarding output platforms (CMS Web Posts, KDP Pamphlets, SSRN Papers).
