# Prompt Skills Index

This directory contains reusable cognitive frameworks, structural formats, and operational system prompts designed to govern AI agent behavior. 

Files are organized using a strict prefixed taxonomy for at-a-glance readability.

## 1. Audits (Verification & Cleaning)
* **`audit_em_dash_cleanup.md`**: Enforces strict punctuation rules, eliminating em-dashes and filler.
* **`audit_fact_integration.md`**: Validates that empirical claims are properly integrated without hallucination.
* **`audit_five_pass_verification.md`**: A rigorous, multi-stage proofing process for final publication drafts.
* **`audit_logical_flow.md`**: Verifies numerical continuity and structural causality throughout a document.

## 2. Formats (Layout & Output)
* **`format_blog_post.md`**: Standardizes Markdown layouts for static CMS/HTML deployment.
* **`format_booklet_printing.md`**: Defines physical pagination and margin constraints for KDP print.
* **`format_carousel_slides/`**: Image assets and layout logic for visual multimodal syndication.

## 3. Workflows (Operational Sequences)
* **`workflow_blog_initialization.md`**: The standard procedural steps for bootstrapping a new article.
* **`workflow_ovp_claude_handoff.md`**: The exact system prompts required when transferring Organic Vernacular Pedagogy tasks to Claude Opus.
* **`workflow_session_opener.md`**: The initialization prompt for establishing operational constraints at the start of a new chat session.

## 4. Frameworks (Cognitive Models)
* **`framework_critical_examination.md`**: A cognitive model for aggressively challenging assumptions and premise flaws.
* **`framework_deconstruct_reconstruct.md`**: A protocol for tearing down weak arguments and rebuilding them systematically.
* **`framework_reasoning.md`**: Core logic directives for agent intelligence and systemic analysis.
