# Visual Prompt Generation Template

## Core Style Directives
- **Aesthetic**: Dramatic realistic creations, authentic human dignity.
- **Prohibitions**: No cartoon aesthetics, no manga styling.
- **Lighting**: Natural, grounded, evoking somber reality.

## Structural Format
`[Subject Context] + [Physical Environment] + [Core Action] + [Lighting/Mood]`

## Transition Mechanics
- Implement slow image fading and morphing.
- Synchronize visual crossfade transition over dubbed audio pause.
- Integrate distinct auditory sigh during morphing sequence.
