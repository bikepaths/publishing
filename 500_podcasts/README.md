# 500_podcasts

Directory manages sequential audio production workflow feeding multimodal pipeline.

## Hierarchy
* **01_scripts**: Text documents containing extracted narrative beats dictating pacing instructions.
* **02_raw_audio**: Unedited voice recordings capturing spoken narrative requiring post-production mastering.
* **03_mastered_tracks**: Finalized audio files possessing perfect pacing pauses feeding multimodal compilation.
* **_archive**: Deprecated podcast assets preventing accidental deployment.
